# office-space

A Waveshare Environment Sensor HAT attached to a Raspberry Pi Zero 2 W records to a local database every minute. It uses an array of sensors to track different conditions.

- BME280: Temperature, Humidity, and Pressure
- TSL25911FN: Ambient Light
- ICM20948: Motion Sensor
- LTR390: UV Sensor
- SGP40: VOC Sensor
