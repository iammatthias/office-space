# office-space

This site display data recored by a Waveshare Environment Sensor HAT attached to a Raspberry Pi Zero 2.

This board contains an array of sensors:

- BME280: Temperature, Humidity, and Pressure
- TSL25911FN: Ambient Light
- ICM20948: Motion Sensor
- LTR390: UV Sensor
- SGP40: VOC Sensor

We record data from these sensors every second to a local database. This site displays the most recent reading from each sensor, along with historic data.
